# TattooAR

**TattooAR** es una aplicación web progresiva (PWA) que permite a los usuarios visualizar tatuajes sobre su piel en tiempo real utilizando **realidad aumentada (AR)**. También conecta a clientes con tatuadores según estilo, ciudad y portafolio.

> Proyecto desarrollado por estudiantes universitarios como solución innovadora para el sector del tatuaje.

---
## Requisitos mínimos del dispositivo

   - Android 10+
   - Cámara con enfoque automático
   - Giroscopio y acelerómetro
   - 3 GB de RAM recomendados
   - Soporte AR

## Tecnologías utilizadas

- **Frontend:** VueJS
- **Backend:** ExpressJS + MongoDB
- **Realidad Aumentada:** Aframe
- **Formato de imágenes:** PNG con fondo transparente
- **Plataforma:** PWA (Aplicación Web Progresiva)
- **API:** RESTful con comunicación segura HTTPS

## Git Flow

Este proyecto usa un flujo de trabajo basado en `Git Flow`:

- **main**: código estable.

- **develop**: desarrollo.

- **feature/***: funcionalidades nuevas.

- **fix/***: corrección de errores.

- **release/***: preparación para nuevos lanzamientos

**[Más información](https://www.atlassian.com/es/git/tutorials/comparing-workflows/gitflow-workflow)**


## Convención de commits

Se usa `Conventional Commits` con estas etiquetas:

- **feat**: funcionalidad nueva

- **fix**: corrección de bugs

- **update**: mejoras menores, documentación o refactor

**[Más información](https://www.conventionalcommits.org/en/v1.0.0/)**
